# Дополнительные изображения для слайдов (опционально)

Для усиления слайдов **8** (маршрутизация / память / trace) и **9** (UX) можно
добавить **обезличенные** скриншоты:

1. **Чат Open WebUI** — одно сообщение с вложением и ответ (без персональных данных, URL, ключей).
2. **DevTools → Network** — фрагмент заголовка ответа с **`X-GPTHub-Trace`** (обрежьте значение до нескольких полей JSON).

После сохранения файлов (например `trace_sample.png`, `chat_sample.png`) можно
вручную вставить их в [`GPTHub_defence_10slides.pptx`](../GPTHub_defence_10slides.pptx)
перед экспортом в PDF, либо расширить `scripts/build_defence_deck_pptx.py`, чтобы
подхватывать картинки из этой папки по соглашению имён.

**Не коммитьте** скриншоты с секретами, внутренними хостами или персональными данными.
